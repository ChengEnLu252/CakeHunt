# 🎂 CakeHunt — 多平台自動化搶蛋糕系統

> 本程式參考自 [Tickets Hunter](https://github.com/bouob/tickets_hunter) 開源專案，
> 並使用 [Claude.ai](https://claude.ai) 協助修改前端風格及擴充多帳號搶票功能。

---

## ✨ 新增功能（相較於原版）

### 🎯 多帳號同時搶票
- 在「自動填表單」頁面底部新增多帳號設定區塊
- 每個帳號各自開啟獨立的瀏覽器視窗同時搶票
- 支援各帳號獨立設定：目標網址、帳號密碼、張數、場次關鍵字、區域關鍵字
- 帳號密碼留空則自動沿用上方全域設定

### 📡 即時搶票監控系統
- 新增「搶票監控」頁面，透過 WebSocket 即時顯示各帳號狀態
- 狀態分類：
  - ① 搶票中
  - ③ 無符合條件，依邏輯順序搶票
  - ④ 搶票成功！請於限定時間內輸入訂單資訊
  - ⑤ 🎉 搶票完成！
  - ②/⑥ 搶票失敗
- 每張帳號卡片顯示即時 Log、當前網址、執行狀態

### 🎨 前端風格改造
- 粉紅可愛主題（Pink Theme）
- 頂部蛋糕裝飾條
- 圓角按鈕、粉紅 Focus 效果
- 支援 Dark Mode

---

## 🚀 使用方式

### 單一帳號搶票
1. 在「基本設定」填入目標售票網址
2. 在「自動填表單」填入帳號密碼
3. 按底部「搶票」按鈕

### 多帳號同時搶票
1. 在「自動填表單」底部「🎯 多帳號搶票設定」新增帳號
2. 每組填入各自的帳號、密碼、目標網址
3. 按底部「多帳號搶票」按鈕啟動
4. 切換至「搶票監控」頁面查看各帳號即時狀態

## 📦 安裝與啟動

pip install -r requirement.txt
python src/settings.py

開啟瀏覽器前往：http://127.0.0.1:16888/settings.html
